# ProjetoCienciaDados

Como rodar o programa
Para executar o programa, primeiro instalar o python no site python.org/downloads
Apos isso, com o powershell aberto no projeto, rodar o comando python -m venv
Apos isso rodar "pip install -r requirements.txt"
Em seguida rodar python app.py
Assim que feito isso ele pode ser aberto no navegador pela "url" localhost:5000
